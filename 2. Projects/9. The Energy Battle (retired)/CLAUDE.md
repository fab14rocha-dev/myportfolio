# The Energy Battle

## What It Is
A local, web-based bike racing game for Matteo. 10 bikes race at the same time on the same screen. Players can pick up hearts for a speed boost or hit bombs that slow them down. First version is single-screen, no server needed.

## Who It's For
Matteo — played locally in the browser.

## Core Game Rules
- 10 bikes race simultaneously
- Hearts on the track = speed boost
- Bombs on the track = slow down
- Race ends when someone crosses the finish line (or a set number of laps)

## Stack
- Vanilla HTML, CSS, JavaScript
- No backend, no server — pure browser game
- Canvas-based rendering (HTML5 Canvas)

## Project Structure
- 1. HTML/ — main game page
- 2. Images/ — sprites, icons, assets
- 3. Scripts/ — game logic JS files
- 5. Styles/ — CSS
