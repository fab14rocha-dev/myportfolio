# Paperboy

## What It Is
A 2D browser-based remake of the classic Paperboy arcade game, built as a learning project by Fabricio.

## Gameplay
- Player rides a bicycle down a street
- Throw newspapers at houses to score points
- Avoid obstacles: cars, dogs, broken glass, etc.
- Lose lives for crashes or missed deliveries

## Stack
- Vanilla JS, HTML Canvas, CSS
- No frameworks, no libraries, no installs

## File Structure
- index.html — game page (just a canvas element)
- css/style.css — basic styles, dark background, centered canvas
- js/main.js — game entry point
- assets/ — sprites, sounds (when added)
- 1. Reference files/ — sketches, notes, references

## Rules
- Keep everything in vanilla JS — no game engines, no libraries
- Use the HTML Canvas API for all rendering
- One JS file per system (player, obstacles, road, etc.) — keep things organized
- Fabricio is a beginner — explain *why* when making decisions, not just *what*
