# VS Code — Workspace Customization

## What it is
VS Code lets you set up your workspace to match how you actually work, not just use the default layout. You can customize extensions, file structure, themes, and settings per project.

## What I learned
Different roles benefit from different setups:
- A developer needs code tools, linters, and debuggers
- A freelancer might need project management + multiple client folders
- A learner benefits from note-taking + a clean, simple layout

CLAUDE.md files are a way to give AI assistants (like Claude Code) permanent context about your project so you don't repeat yourself every session.

## Key ideas
- Keep one workspace folder per project or area
- Use a numbered folder structure to control sort order (1. Learning, 2. Projects, etc.)
- Settings can be workspace-specific (`.vscode/settings.json`) so they don't affect other projects
